## Project Overview

**rain_edit** is a poetic, atmospheric code/text editor that runs “off in the rain.” It’s a creative, moody take on the classic coding environment.

- **Core Concept**: A beautiful, rainy-day editor with strong aesthetic and emotional atmosphere.
- **Vibe**: Calm, reflective, almost meditative coding experience.

## Technology Stack

- JavaScript / TypeScript
- Likely WebGL/WebGPU or Canvas for rain effects + editor
- Text editing core (possibly CodeMirror, Monaco, or custom)

## Grok Guidelines

- **Atmosphere First**: The rain, lighting, sound, and mood are the main features — prioritize beauty and immersion.
- **Usability**: It still needs to be a functional editor (syntax highlighting, editing, saving, etc.).
- **Creative Extensions**: Rain intensity linked to typing speed, sound design, different weather moods, or time-of-day themes would be powerful.
- **Performance**: Keep the rain effects smooth even while editing large files.
